# ROS Actions

ros_actions klasoru içerisinde python programlama dili kullanılarak yapılmış bir örnek mevcuttur.

Kullanım örneği:

  1- 1.Terminal 
  ```
  roscore
  ```   
  
  2- 2.Terminal 
  ```
  rosrun ros_basic_tutorial python_action_client.py
  ```
  
  3- 3.Terminal
  ```
  rosrun ros_basic_tutorial python_action_server.py
  ```
